# 김영진 💛 구경림 가족 일기

하나님을 사랑하는 우리 가족의, 주님께 인도받는 시간들의 기록.
로그인한 가족만 볼 수 있는 **비공개 가족 일기 · 캘린더** 웹앱입니다.

- **프론트**: 순수 HTML/CSS/JS (GitHub Pages 무료 호스팅)
- **데이터·로그인·사진**: Supabase (무료) — 로그인(인증)한 가족만 접근(RLS)
- 부부가 **PC·모바일 어디서나** 웹에서 직접 기록·수정, 실시간 공유

---

## 사용법 (배포 후)

1. 사이트 주소 접속 → **가족 이메일로 로그인**
2. **📜 타임라인**: 오른쪽 아래 **＋** → 연대기 기록 추가(날짜/기간·카테고리·구성원·사진). 사진은 올리면 자동 최적화 + ✂️영역으로 썸네일 지정.
3. **🗓️ 캘린더**: ＋ 또는 날짜 클릭 → 일정 추가. **‘종류/반복’** 에서 `일정(한 번)` / `매주·매월` / `🎂 기념일(매년, 음력 지원)` 선택.
4. **🔍 검색**: 제목·내용·구성원으로 찾기.
5. **⬇️ 엑셀 백업**: 우측 상단 버튼으로 전체 데이터를 내 PC에 저장.

> 데이터는 모두 Supabase 클라우드에 저장됩니다. 추가/수정은 **재배포 없이** 바로 반영돼요.

## 설정 (`config.js`)
- `siteTitle` / `siteSubtitle`: 제목·부제
- `counters`: 헤더 카운터(결혼·아이 생일 등 기준일)
- `categories` / `scheduleCategories`: 카테고리(이모지·색)
- `members`: 구성원 체크박스 목록
- `photo`: 사진 최적화 크기·품질
- `supabaseUrl` / `supabaseAnonKey`: Supabase 연결 (anon 키는 공개용이라 안전)

## 배포 갱신
코드를 바꾼 뒤 `git add -A && git commit -m "..." && git push` 하면 사이트가 갱신됩니다.

## 참고
- Supabase 스키마·시드 SQL은 `supabase/` 폴더에 있으며, 공개 저장소에는 올라가지 않습니다(.gitignore).
- 사진/데이터 백업 원본(`events.xlsx`, `photos/` 등)도 로컬 전용입니다.
